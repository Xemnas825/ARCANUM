# ARCANUM - Backend

Backend de la aplicación D&D construida con **TypeScript**.

## Responsabilidades
- API REST para gestión de personajes
- Almacenamiento de datos (razas, clases, personajes)
- Validación de reglas de D&D
- Autenticación y sesiones

## Configuración (próximamente)
- Node.js + Express/Fastify
- TypeScript
- Base de datos (MongoDB/PostgreSQL)
- Validación de datos
